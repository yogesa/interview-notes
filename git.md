# Git

git 是一个分布式的版本控制工具.

### 如何让自己的主机在分布式网络中确认身份?

```git config```   ```--global ```这台机器上的所有仓库都会使用这个配置

通过 ```git config```   ```--global ``` 配置主机

```shell
$git config --global user.name "chenhui"
$git config --global user.email "1316238389@qq.com"
```

### 使用Git

在想要要建立仓库的目录下, 初始化仓库

```shell
$ git init
```
